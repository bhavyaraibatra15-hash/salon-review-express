## 📖 Overview

JazzUp Review Assistant is a modern AI-powered web application designed to help customers express their real salon experience naturally and professionally.

Instead of writing reviews from scratch, customers simply answer a few questions about their visit, and the AI generates multiple authentic review options that they can edit, copy, and post to Google.

The application is designed with a premium, mobile-first user experience and can easily be customized for any local business.

## 📱 User Flow

1. Customer selects their visit details.
2. AI securely generates review options.
3. Three review variations are displayed.
4. Customer copies the preferred review.
5. Google Review page opens automatically.
6. Customer pastes the review and submits.

Average completion time:
**Less than 60 seconds**

## 🎯 Main Features

### Review Generator

Generate three unique review versions based only on the customer's actual experience.

### Smart AI

The AI never invents information.

It only uses customer-selected details to generate genuine reviews.
